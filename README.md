# 🏋️‍♂️ FIT LIVE

O **FIT LIVE** é um projeto web desenvolvido com foco em saúde, musculação e incentivo a hábitos saudáveis.  
O sistema oferece cálculo de IMC, informações sobre exercícios físicos e uma área voltada para acompanhamento profissional.

---

## 📌 Sobre o Projeto

Este projeto foi criado com o objetivo de praticar desenvolvimento web utilizando HTML, CSS e JavaScript, além de aplicar conceitos de organização de layout, responsividade e interatividade.

O site é dividido em páginas que abordam saúde, cálculo corporal e personalização de treinos.

---

## 🚀 Funcionalidades

- ✅ Página inicial com apresentação sobre saúde e exercícios físicos  
- ✅ Calculadora de IMC com entrada de peso, altura e idade  
- ✅ Exibição automática do resultado do IMC  
- ✅ Área para inserção de descrição médica e exercícios personalizados  
- ✅ Carrossel automático de imagens  
- ✅ Seções organizadas: Home, Musculação, Cálculo e Nutrição  
- ✅ Rodapé com mapa e ícones de redes sociais
